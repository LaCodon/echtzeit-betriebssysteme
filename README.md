# Pflanzenbewässerung

Autoren: Tim Schmidt und Fabian Maier

Belegarbeit im Fach Echtzeitbetriebssysteme an der FH Erfurt im 1. Semester Master Angewandte Informatik

## Aufbau des Repositories

Im Verzeichnis `ui` befindet sich der Code für das Benutzerinterface. In `gpio` befindet sich die echtzeit Steuerungssoftware und unter `benchmarks` sind cyclic-tests mit PREEMT_RT Patch und ohne für den Linux Kernel abgelegt.
